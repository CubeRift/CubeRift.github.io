What is CodeVault?

CodeVault is a free, open code library built for engineering students. It lets you store your lab programs, access them from any device, and browse code uploaded by others — all without digging through old WhatsApp messages or losing files before exams.

The public library requires no login. Anyone can browse, search by language, and copy code instantly. Students with an account can upload and organise their own private collection using folders.

Everything is stored securely using GitHub Gists — so your code is always safe, always accessible, and never tied to a single device.

How It Works

CodeVault uses GitHub Gists as its storage backend — a free, reliable service by GitHub. Your code is stored as private gists under our account, accessible only through CodeVault's interface.

The frontend is a single HTML file with no frameworks or heavy dependencies — so it loads fast even on 4G. Authentication uses SHA-256 password hashing in the browser, meaning your plain-text password is never transmitted or stored anywhere.

The public library fetches only publicly tagged snippets and renders them client-side — no server, no database, no maintenance headaches.
